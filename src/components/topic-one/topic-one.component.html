*ngIf & @if

*ngIf Directive (Legacy) @if Block (Modern)
Requires importing CommonModule or NgIf into your component. Built directly into the template engine. No imports needed.
Must be attached to a DOM element (like a <div>) or an <ng-template>. Independent of DOM elements. You can wrap plain text directly without creating extra HTML tags.
Handling "else" conditions is clunky and requires referencing local template variables (; else myTemplate). Highly readable and intuitive syntax with native @else if and @else blocks.
