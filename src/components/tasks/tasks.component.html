To Do List App
  • Create a text box to enter a new task.
  • On clicking Add, save the task to a local list and persist it in localStorage.
  • Load and display existing items from localStorage on page load.
  • Checking a task's box toggles its status between completed and pending.
  • Provide inline controls to edit task names or delete tasks completely.
Code Hint

LocalStorage Persistence (TS)

// Save to storage
saveToLocalStorage() {
  localStorage.setItem(
    'todo-list', 
    JSON.stringify(this.todos)
  );
}

// Load from storage
ngOnInit() {
  const stored = localStorage.getItem('todo-list');
  if (stored) this.todos = JSON.parse(stored);
}
